# Яндекс Музыка — Веб-интерфейс

Графический интерфейс для управления библиотекой Яндекс Музыки: просмотр, поиск и скачивание треков.

## Быстрый старт

### macOS

Дважды кликните по файлу **`start.command`** в корне проекта. Всё установится и запустится автоматически.

### Windows

Дважды кликните по файлу **`start.bat`** в корне проекта.

### Ручной запуск

```bash
# 1. Установите зависимости (один раз)
pip install flask browser_cookie3 requests typing_extensions aiohttp aiofiles

# 2. Запустите сервер
python gui/app.py

# 3. Откройте в браузере
# http://127.0.0.1:5000
```

## Авторизация

### Автоматический вход (рекомендуется)

1. Войдите в свой аккаунт на [music.yandex.ru](https://music.yandex.ru) в браузере
2. Откройте GUI и нажмите **«Войти автоматически»**
3. Токен будет получен из cookies браузера автоматически

Поддерживаемые браузеры: Chrome, Firefox, Edge, Opera.

### Вход по токену

Если автоматический вход не работает, вставьте OAuth-токен вручную. Подробнее о получении токена — в [документации](../docs/source/token.md).

## Возможности

### Мне нравится
- Просмотр всех лайкнутых треков с обложками
- Скачанные треки отмечены зелёной галочкой
- Пакетное скачивание: выберите треки и нажмите «Скачать выбранные»
- Кнопка «Скачать все» — загрузит всю библиотеку

### Плейлисты
- Просмотр всех плейлистов
- Открытие плейлиста со списком треков
- Скачивание треков из плейлиста

### Поиск
- Поиск по трекам, альбомам и артистам
- Просмотр альбомов и треков артиста
- Скачивание найденных треков

### Загрузки
- Список всех скачанных файлов
- Размер каждого файла
- Скачивание файлов на устройство

### Настройки
- Выбор папки для загрузок (визуальный файл-пикер или ручной ввод)
- Настройки сохраняются между сессиями

## Структура

```
gui/
├── app.py              # Flask-сервер (API + роуты)
├── templates/
│   └── index.html      # HTML-разметка
└── static/
    ├── style.css       # Стили (тёмная тема)
    └── app.js          # Логика интерфейса
```

## Технические детали

- **Backend**: Flask (Python)
- **Frontend**: Vanilla HTML/CSS/JS (без фреймворков)
- **Авторизация**: `browser_cookie3` для чтения cookies + Yandex Passport API
- **Сессия**: токен сохраняется в `gui/.gui_config.json` (в `.gitignore`)
- **Скачивание**: MP3, 192 kbps по умолчанию, фоновая загрузка с прогресс-баром
