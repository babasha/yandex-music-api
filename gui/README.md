# Яндекс Музыка — Веб-интерфейс

Графический интерфейс для управления библиотекой Яндекс Музыки: просмотр, поиск и скачивание треков.

## Быстрый старт

### macOS

1. Дважды кликните по файлу **`start.command`** в Finder
2. Готово — браузер откроется автоматически

> При первом запуске скрипт сам установит все зависимости.

### Windows

1. Дважды кликните по файлу **`start.bat`**
2. Готово — браузер откроется автоматически

> Требуется установленный [Python 3.8+](https://www.python.org/downloads/). При установке поставьте галочку **«Add Python to PATH»**.

### Ручной запуск

```bash
# 1. Установите зависимости (один раз)
pip install flask browser_cookie3 requests typing_extensions aiohttp aiofiles

# 2. Запустите сервер
python gui/app.py

# 3. Откройте в браузере
# http://127.0.0.1:5000
```

## Авторизация

### Автоматический вход (рекомендуется)

1. Войдите в свой аккаунт на [music.yandex.ru](https://music.yandex.ru) в любом браузере
2. Откройте GUI и нажмите **«Войти автоматически»**
3. Токен будет получен из cookies браузера — никаких ручных действий

Поддерживаемые браузеры: **Chrome**, **Firefox**, **Edge**, **Opera**.

Сессия сохраняется — после перезагрузки страницы или перезапуска сервера повторный вход не нужен.

### Вход по токену

Если автоматический вход не работает, вставьте OAuth-токен в поле на экране входа.

Как получить токен — см. [инструкцию](../docs/source/token.md).

## Возможности

### Мне нравится
- Просмотр всех лайкнутых треков с обложками и длительностью
- Скачанные треки отмечены зелёной галочкой и подсвечены
- Выбор треков чекбоксами + кнопка **«Скачать выбранные»**
- Кнопка **«Скачать все»** — загрузит всю библиотеку
- Пагинация по 50 треков

### Плейлисты
- Сетка плейлистов с обложками
- Открытие плейлиста со списком треков
- Скачивание треков из любого плейлиста

### Поиск
- Поиск по трекам, альбомам и артистам
- Просмотр содержимого альбомов
- Просмотр треков артиста
- Скачивание найденных треков в один клик

### Загрузки
- Список всех скачанных файлов с размерами
- Прогресс-бар при пакетной загрузке (название текущего трека, счётчик, процент)

### Настройки
- Выбор папки для загрузок — визуальный файл-пикер или ручной ввод пути
- Все настройки сохраняются между сессиями

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/api/auto-login` | Автоматический вход через cookies браузера |
| `POST` | `/api/login` | Вход по OAuth-токену |
| `POST` | `/api/logout` | Выход |
| `GET` | `/api/auth-status` | Проверка авторизации |
| `GET` | `/api/likes/all` | Лайкнутые треки (пагинация) |
| `GET` | `/api/playlists` | Список плейлистов |
| `GET` | `/api/playlist/<kind>` | Треки плейлиста |
| `GET` | `/api/search` | Поиск (треки/альбомы/артисты) |
| `GET` | `/api/album/<id>` | Альбом с треками |
| `GET` | `/api/artist/<id>/tracks` | Треки артиста |
| `POST` | `/api/download` | Скачать один трек |
| `POST` | `/api/download/batch` | Пакетная загрузка |
| `GET` | `/api/download/progress/<id>` | Прогресс загрузки |
| `GET` | `/api/downloads` | Список скачанных файлов |
| `GET/POST` | `/api/settings` | Настройки (папка загрузок) |
| `POST` | `/api/browse` | Навигация по папкам (файл-пикер) |

## Технические детали

- **Backend**: Flask (Python)
- **Frontend**: Vanilla HTML/CSS/JS — без фреймворков, работает в любом браузере
- **Авторизация**: `browser_cookie3` читает cookies с диска + обмен через Yandex Passport API
- **Сессия**: токен сохраняется локально в `gui/.gui_config.json` (добавлен в `.gitignore`)
- **Скачивание**: MP3 192 kbps, фоновая загрузка в отдельном потоке
- **Тема**: тёмная, в стиле Яндекс Музыки
